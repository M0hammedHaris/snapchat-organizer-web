# Snapchat Organizer Web

A modern, responsive web landing page for the [Snapchat Organizer Desktop](https://github.com/M0hammedHaris/snapchat-organizer-desktop) application.

## Overview

This web application serves as the landing page for Snapchat Organizer - a powerful desktop tool to export, organize, and preserve your Snapchat memories with advanced metadata tools and a beautiful interface.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/M0hammedHaris/snapchat-organizer-web.git
cd snapchat-organizer-web
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
